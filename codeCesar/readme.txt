# Chiffrement de César en C++

Ce projet est une implémentation simple du chiffrement de César en C++. Le chiffrement de César est une technique de chiffrement très simple utilisée par Jules César dans ses correspondances secrètes (d'où le nom).

## Description

Le programme prend un message et une clé en entrée, convertit le message en majuscules et renvoie le message chiffré en utilisant le chiffrement de César.

### Utilisation

Lorsque vous exécutez le programme, il vous demandera d'entrer le message à chiffrer et la clé de César. La clé est le nombre de décalages dans l'alphabet que vous souhaitez appliquer à chaque lettre du message.

Par exemple, si vous entrez le message "HELLO" avec la clé 3, le programme renverra "KHOOR", qui est le message chiffré.
